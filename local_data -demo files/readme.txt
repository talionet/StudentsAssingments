This dir contains a demo version of the files used for 'StudentsAssignment', with StudentsID deleted.

pretest_scores_[date].csv - latest pretest scores Sent via mail from CET when there is an update in pretest_scores. 
MD_classes_[date].csv - meta-data on calsses in the experiment. Including start dates (updated manualy), % pretest scores (updated automatically based on pretest-scores file.
MD_users_[TimeStamp].csv -  meta-data on students + assignments to experimental group. Updated when pretest scores are updated
MD_users_test.xlsx - An excel file to ensure there is no mistake in MD_user update.
